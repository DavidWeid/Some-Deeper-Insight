Rover Reddit

Rover Reddit is a full stack web application that uses Node, Express, React, Cheerio, and Passport. Users can sign on to share, comment, and save photos from NASA's API and articles scraped from space.com.

Rover Reddit was built by two developers meeting weekly for standups, code review, and discussions about what was done, was is being done, and what will be done.

This application is one of the few places to browse Mar's Rover images directly from NASA - a largely resource due to the lack of a friendly UI.

Rover Reddit connects Mars to the User.
